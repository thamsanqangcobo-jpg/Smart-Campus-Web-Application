Smart Campus Web Application
Overview

The Smart Campus Web Application is a web-based system designed to improve communication, safety, and collaboration within a campus environment. The platform allows students and staff to interact through real-time messaging and provides tools for reporting incidents or emergencies directly to campus authorities.

The system was developed using ASP.NET Core and modern web technologies, demonstrating concepts such as MVC architecture, real-time communication, database integration, and secure user authentication.

Key Features
User Authentication

Secure user registration and login

Session-based authentication

Role-based access for different users (students or staff)

Real-Time Chat System

Live messaging between users

Built using SignalR for real-time communication

Enables fast and reliable communication within the campus

SOS / Incident Reporting

Users can submit emergency reports

Reports include details such as:

Incident type

Description

Location

Date reported

Allows administrators to monitor and respond to incidents

Incident Management

Track reported incidents

Update incident status

View active or resolved reports
